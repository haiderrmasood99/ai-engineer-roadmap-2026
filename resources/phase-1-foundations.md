# Phase 1: Foundations

Outcome: become dangerous with Python, data, APIs, and enough math to debug models.

## Learn

- Python packaging, typing, tests, and CLI scripts
- SQL and data modeling basics
- REST APIs and background jobs
- linear algebra, probability, derivatives, and optimization intuition
- Git, GitHub Actions, and reading other people's code

## Build

Build a CLI data pipeline that ingests messy CSV files, validates schema, writes clean outputs, and has tests.

## Resources

- Python docs: packaging, pathlib, dataclasses, typing
- FastAPI docs for API fundamentals
- Khan Academy or 3Blue1Brown for linear algebra intuition
- Practical SQL exercises with real datasets

## Skip

- memorizing every math proof before building
- copying notebooks without turning them into software
