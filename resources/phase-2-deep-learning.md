# Phase 2: Deep Learning

Outcome: train, debug, and explain neural networks without treating them like magic.

## Learn

- PyTorch tensors, datasets, dataloaders, training loops
- CNNs, transfer learning, segmentation basics
- loss functions, regularization, overfitting, calibration
- experiment tracking and reproducibility
- error analysis by slice, not just aggregate accuracy

## Build

Build an image classifier with preprocessing, Grad-CAM explainability, and a failure analysis report.

## Resources

- PyTorch tutorials
- fast.ai practical deep learning
- Sebastian Raschka's ML and deep learning materials
- Papers with code implementations, but only after reproducing basics

## Skip

- chasing architecture names before understanding data leakage
- leaderboard thinking without deployment constraints
